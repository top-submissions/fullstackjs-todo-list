# Solo Project Template - Default

A streamlined template repository for individual web development projects with essential documentation and clean project structure.

## üöÄ Quick Start

### Prerequisites
- Modern web browser
- Code editor (VS Code recommended)
- Git (for version control)

### Getting Started
1. Use this template or clone the repository:
   ```bash
   git clone https://github.com/your-username/templates-solo-default.git
   cd templates-solo-default
   ```

2. Open the project in your browser:
   - Directly open `src/index.html` in a browser, or
   - Use a local server for better functionality:
     ```bash
     # Using Python
     python -m http.server 8000
     
     # Using Node.js with http-server
     npx http-server src
     ```
